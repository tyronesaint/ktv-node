# 音源支持说明

## 支持的音源列表

### ✅ 酷我音乐 (kw) - **完全支持**

**状态**：无需加载脚本，直接可用

**支持功能**：
- ✅ 搜索歌曲
- ✅ 获取播放链接
- ✅ 支持多种音质（128k, 320k, flac）

**使用方法**：
1. 在"选择音源"中选择"kw"
2. 在搜索框中输入关键词（如"周杰伦"）
3. 点击"搜索"按钮
4. 从结果中选择歌曲
5. 点击"获取链接"即可

**优势**：
- 快速响应（< 2秒）
- 无需加载脚本
- 稳定可靠

---

### ⚠️ 网易云音乐 (wy) - **需要加载脚本**

**状态**：需要先加载音源脚本才能使用

**支持功能**：
- ✅ 搜索歌曲（需要脚本）
- ✅ 获取播放链接（需要脚本）
- ✅ 支持多种音质（128k, 320k, flac, flac24bit, hires, atmos, master）

**使用方法**：
1. 在"1. 加载音源脚本"部分，点击"📥 加载脚本"按钮
2. 等待脚本加载完成（需要几秒钟）
3. 在"选择音源"中选择"wy"
4. 在搜索框中输入关键词（如"稻香"）
5. 点击"搜索"按钮
6. 从结果中选择歌曲
7. 点击"获取链接"即可

**注意事项**：
- 必须先加载脚本，否则会提示"请先加载音源脚本"
- ikun API 使用自签名 SSL 证书，可能需要重试几次
- 搜索和获取链接速度较慢（5-10秒）

**常见问题**：
- **搜索失败**：检查是否已加载脚本
- **获取链接失败**：可能是因为 ikun API 证书问题，可以重试
- **一直转圈**：可能是因为 API 超时，请检查网络连接

---

### ❌ Git Music (git) - **暂不支持**

**状态**：搜索和获取链接暂不支持

**说明**：
- Git Music 是洛雪音乐的第三方音源
- 目前搜索功能暂未实现
- 未来可能会添加支持

## 技术细节

### 酷我音乐实现原理

酷我音乐使用直接 API 调用：

```javascript
// 搜索 API
http://search.kuwo.cn/r.s?all={关键词}&pn=0&rn=20&ft=music&rformat=json&encoding=utf8&vipver=1

// 获取 URL API
http://antiserver.kuwo.cn/anti.s?type=convert_url&rid={歌曲ID}&format=mp3&response=url
```

### 网易云音乐实现原理

网易云音乐使用洛雪音乐音源脚本（ikun）：

1. 加载脚本：`https://ghproxy.net/https://raw.githubusercontent.com/pdone/lx-music-source/main/ikun/latest.js`
2. 脚本初始化：模拟洛雪音乐的 `lx` 全局对象
3. 调用脚本API：通过 `lx.request()` 调用 ikun API
4. 获取结果：解析响应并返回播放链接

## 故障排除

### 酷我音乐问题

**问题1：搜索失败**
- 解决：检查网络连接，确保可以访问酷我音乐 API

**问题2：获取链接失败**
- 解决：可能是歌曲 ID 无效或歌曲已下架，尝试其他歌曲

### 网易云音乐问题

**问题1：提示"请先加载音源脚本"**
- 解决：先点击"📥 加载脚本"按钮，等待加载完成

**问题2：脚本加载失败**
- 解决：
  - 检查网络连接
  - 检查 GitHub 访问是否正常
  - 尝试使用 ghproxy.net 代理（已配置）

**问题3：搜索一直转圈**
- 解决：
  - 可能是 ikun API 证书问题，稍后重试
  - 检查网络连接
  - 尝试使用酷我音乐替代

**问题4：获取链接失败**
- 解决：
  - 确保已加载脚本
  - 检查歌曲 ID 是否有效
  - 可能是 ikun API 临时故障，稍后重试

## 常见问题 FAQ

### Q1: 为什么酷我音乐不需要加载脚本？
A: 酷我音乐的 API 是公开的，可以直接调用，无需通过音源脚本。

### Q2: 网易云音乐为什么需要加载脚本？
A: 网易云音乐的 API 需要加密和签名，音源脚本封装了这些复杂的逻辑。

### Q3: ikun API 的 SSL 证书问题是怎么回事？
A: ikun API 使用自签名 SSL 证书，这是洛雪音乐音源的标准做法。服务器已经配置为禁用证书验证。

### Q4: 为什么有些音源不支持搜索？
A: 搜索功能需要音源脚本提供对应的 `search` action，并非所有音源都支持。

### Q5: 未来会添加更多音源吗？
A: 计划中！未来会根据需求添加更多音源的支持。
